Introducing Asynchronous Generators

To demonstrate, let's invent a literal syntax for Observables. This literal syntax is not being proposed. It is used only for the purposes of visualization. 
// A sequence of numbers arriving over time...
var someObservable = <|...1....2....3...|>; // for the last time, not proposing this syntax. :-)
someObservable.observe({
    next: function(v) {
        console.log(v);
    },
    return: function() {
        console.log("done");
    }
}
});
If run, this code would log the following... 
> immediately after observe
> 1
> 2
> 3
> done
...with each number and the "done" message being logged an equal amount of time apart.
